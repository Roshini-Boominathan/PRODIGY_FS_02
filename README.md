# Employee Management System

A Java-based web application using Java Servlets, JDBC, JSP, and MySQL.

## Features
- Add Employee
- View Employees
- Delete Employee
- MVC Architecture

## Technologies
- Java
- JDBC
- JSP & Servlets
- MySQL
- Apache Tomcat

## How to Run
1. Import project into Eclipse
2. Add MySQL connector to WEB-INF/lib
3. Configure database
4. Run on Tomcat server
